Correlation heatmap







For analysis we will be using variables that are correlated with the outcome variable. This correlation plot is similar to a symmetric matrix but it immediately allows us to point out the most correlated and anti-correlated atributes.

It allows us to see what we need to investigate and which varaibles we can focus on if we do not have the time and resources to investigate every possible column.

The (color) indicates high values for my correlation and (color) indicate low values for correlation.
